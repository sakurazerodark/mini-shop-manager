# 店小易 - 小店商品管理与收银系统

> 专为一人店、夫妻店设计的小型商品管理和收银系统，轻量、简单、易用。

## 项目简介

**店小易**是一款面向小型零售店铺（如便利店、杂货店、夫妻店等）的轻量级商品管理与收银系统。采用桌面应用形式，数据本地存储，无需复杂的服务器部署，开箱即用。

## 技术栈

- **前端**: Vue 3 + Vite
- **桌面端**: Electron
- **后端**: Express.js
- **数据库**: SQLite3
- **UI**: 原生 CSS，响应式设计

## 功能特性

### 已完成功能

- **收银台 (POS)**
  - 条码扫描录入商品
  - 购物车管理（增删改数量）
  - 称重商品支持
  - 多种支付方式（现金、支付宝、微信支付、云闪付）
  - 订单结算与打印

- **商品与库存管理**
  - 商品信息录入与编辑
  - 条码查询（支持第三方商品码 API）
  - 库存管理与预警
  - 供应商管理
  - 商品图片上传

- **销售报表**
  - 营业大盘统计
  - 订单明细查询
  - 支付记录查看

- **系统设置**
  - 支付方式配置
  - 支付宝当面付（沙箱环境已跑通）
  - 微信支付 V3 配置
  - 云闪付配置
  - 商品码 API 配置

## 快速开始

### 环境要求

- Node.js >= 18
- npm 或 yarn

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run start
```

这将同时启动 Vite 开发服务器和 Electron 应用。

### 构建生产版本

```bash
npm run build
```

## 配置说明

### 支付宝配置

1. 登录支付宝开放平台
2. 进入沙箱环境获取测试账号
3. 在系统设置中填写：
   - 应用 ID (APP ID)
   - 支付宝网关
   - 商户私钥
   - 支付宝公钥

### 微信支付配置

1. 登录微信支付商户平台
2. 获取商户号、API 证书等
3. 在系统设置中填写相关配置

### 商品码 API 配置

支持探数 API 等第三方商品条码查询服务，配置 API Key 后可自动获取商品信息。

## 注意事项

- 当前版本仅支付宝沙箱环境已跑通，生产环境使用前请仔细测试
- 微信和云闪付支付功能尚未完成联调，请勿直接用于生产
- 商品码 API 需要自行申请并配置
- 建议定期备份 `data/shop.db` 数据库文件
